What is a documentation collection?

A documentation collection is a logically and structurally consistent set of data files.

Each collection requires different documentation information so you may have to complete this form more than once.

What about documents and images? You could document these as a single collection; however, we ask that you provide a brief description of each collection. This will be used across the collection. In the case of documents and images, this description servers a similar role as a poster or paper abstract. We strongly encourage you to provide a separate collection for each document.

You can create new collections and edit existing, unsubmitted collections here.
