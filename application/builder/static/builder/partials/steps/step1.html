What kind of data files are described by this collection? (Remember, if the data files are structurally different, you need to create multiple collections for documentation.)

  • Documents (PDF, Word, RTF, etc), Images (PNG, Tiff, JPEG, etc), Videos (URLs)
  • Spreadsheets (Excel, ODF, etc), CSV, TSV, DAT*,
  • Shapefiles, KML or other vector format
  • Rasters (GeoTiff, MrSID, ASCII), NetCDF, HDF 4 or 5
  • All other formats - proprietary data formats**, model executables, 2D/3D matrices***, database (geodatabase, MS Access, SQLite, etc), web services
